# PowderPlan

AI-powered ski trip planner. Enter your trip dates, group size, budget, and vibe — get AI-generated resort recommendations with real flight options, live snow conditions, and hotel listings.

## Stack

- **Frontend**: React + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Supabase (Postgres + Edge Functions)
- **AI**: Gemini 2.5 Flash
- **APIs**: Amadeus (flights + hotels), Wikipedia (resort photos)

## Dev

```sh
bun install
bun dev
```
